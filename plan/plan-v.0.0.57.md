# 편한가계부 후속 개발 진행 기록 v.0.0.57

작성일: 2026-06-26

## 1. 기준

- 이전 기록: `plan/plan-v.0.0.56.md`
- 이번 기록 범위: v0.0 개발 라인 종결 처리
- 사용자 요청: `plan-v.0.0.*` 기반 개발사항을 종결하고 `plan-v.0.1.*` 신규 개발 라인을 생성

## 2. v0.0 개발 라인 종결

v0.0 라인은 MVP 기능 구현, 백엔드 서비스 분리, 프론트엔드 화면·hook 분리, 검색 페이지네이션, 영수증 파일 정리 정책까지의 개발 사항을 기준으로 종결한다.

이후 신규 기능 개발은 `plan-v.0.1.*` 라인에서 관리한다.

### v0.0 최종 완료 범위

- 월별 거래 목록, 검색, 필터, 상세, 수정, 삭제
- 수입/지출/이체 등록 및 자산 잔액 반영
- 카테고리, 자산, 명의, 예산 관리
- 통계, 월별/연별 요약, 주간 지출, 예산 사용 현황
- 공동/개인 소비 구분과 소비 명의 관리
- 카드 결제 예약, 반복 거래, 할부 거래
- 영수증 다중 첨부, 조회, 미리보기, 삭제
- 거래 삭제 시 영수증 파일 정리
- 클립보드 카드/은행 문자 자동 분석 및 거래 입력
- 검색 결과 페이지네이션과 정렬
- 현재 검색 페이지 CSV 내보내기 범위 명확화
- 백엔드 `LedgerService` 제거 및 도메인별 서비스 분리
- 프론트엔드 App 책임 축소, API 모듈과 hook 분리

## 3. v0.0에서 이월하지 않는 항목

아래 항목은 v0.0 라인에서 추가 진행하지 않고, 필요 시 v0.1 이후 요구사항에 맞춰 재평가한다.

- 검색 조건 전체 결과 CSV 내보내기 API
- 대량 거래 검색 UX 개선
- 기존 orphan 영수증 파일 정리 운영 도구
- v0.0 범위의 백엔드 서비스별 추가 통합 테스트

## 4. v0.1 신규 개발 방향

v0.1 라인의 주 개발 주제는 영수증 OCR 기반 자동 입력이다.

세부 시작 문서는 `plan/plan-v.0.1.0.md`에서 관리한다.
