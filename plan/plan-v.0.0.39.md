# 편한가계부 후속 개발 진행 기록 v.0.0.39

작성일: 2026-06-25

## 1. 기준

- 이전 기록: `plan/plan-v.0.0.38.md`
- 이번 개발 범위: v.0.0.39 권장 항목 1순위인 연간 예산 연도 이동과 주간별 통계
- 검증 방식: Gradle 전체 테스트, Docker Compose 이미지 빌드, Playwright 브라우저 스모크 테스트

## 2. 이번 버전 개발 완료

### 연간 예산 요약 API

- `GET /api/budgets/summary/yearly?year={year}` API를 추가했다.
- 선택 연도의 12개월 월 예산과 실제 지출을 함께 집계한다.
- 연간 예산, 연간 지출, 남은 예산, 사용률을 제공한다.
- 월별 예산·지출·잔여 금액·사용률·초과 여부를 1월부터 12월까지 반환한다.
- 예산을 설정하지 않은 달도 0원 예산으로 포함해 연간 흐름이 끊기지 않도록 했다.

### 예산 화면 월별·연간 전환

- 통계의 예산 탭에 `월별`, `연간` 기간 전환을 추가했다.
- 연간 예산 화면은 월 이동 대신 이전 연도·다음 연도 이동을 사용한다.
- 선택 연도의 연간 예산 사용률과 12개월 예산 대비 지출을 표시한다.
- 월 예산을 초과한 달은 강조 색으로 표시한다.
- 기존 월별 예산 설정과 전월 복사 흐름은 그대로 유지한다.

### 월별 주간 지출 통계

- 월 요약 응답에 주간 지출 목록을 추가했다.
- 각 주는 월 시작일부터 일요일까지, 이후 월요일부터 일요일까지 집계하며 마지막 주는 월 말일에서 종료한다.
- 주차별 시작일, 종료일, 지출 합계, 지출 거래 건수를 제공한다.
- 월별 예산 화면 하단에 주간 지출 금액과 상대 막대, 기간·거래 건수를 표시한다.

### 연도 탐색 공통화

- 연별 통계와 연간 예산에서 연도 전용 탐색 UI를 사용하도록 했다.
- 연간 화면에서 이전·다음 조작 시 1개월이 아닌 1년 단위로 이동한다.

### 테스트 및 스모크 보강

- 월 거래를 일요일 종료 기준 주차로 분리하는 단위 테스트를 추가했다.
- 연간 예산과 지출, 월별 초과 상태를 집계하는 단위 테스트를 추가했다.
- 브라우저 스모크 테스트에서 주간 지출 영역, 연간 예산 API, 12개월 목록, 연도 탐색 UI를 검증한다.

## 4. 아직 부분 완료인 항목

### 검색·필터

- 월 내 클라이언트 필터와 사용자 지정 기간 거래 조회는 구현되어 있다.
- 금액 범위, 자산, 다중 조건을 서버에서 조합하는 고급 검색·필터는 아직 미완료다.

### 프로젝트 구조

- 프론트엔드 주요 화면과 상태가 `frontend/src/main.jsx`에 집중되어 있다.
- 백엔드 `LedgerService`가 거래·예산·명의·통계 책임을 함께 담당하고 있어 서비스 분리가 필요하다.

### 기타 후속 항목

## 7. 개발 메모

- 연간 예산은 별도의 연 예산 값을 저장하지 않고 12개월 월 예산 합계로 계산한다.
- 예산이 0원인 달의 사용률은 0%로 표시하되 실제 지출 금액은 월 목록에 그대로 표시한다.
- 주간 경계는 달력 주간 흐름을 읽기 쉽도록 일요일 종료 기준으로 정했다.
