## 2. 이번 버전 개발 완료

### 검색 CSV 범위 명확화

- 검색 필터가 활성화된 상태에서 CSV 버튼 문구를 `현재 페이지 CSV`로 변경했다.
- 통계/요약 화면의 필터 CSV 버튼 문구도 `현재 검색 페이지 CSV 내보내기`로 변경했다.
- 검색 필터 영역에 `CSV는 현재 페이지에 표시된 N건만 내보냅니다.` 안내 문구를 추가했다.
- 검색 페이지 상태가 있는 CSV 파일명에 `page-{현재쪽}-of-{전체쪽}` suffix를 붙이도록 했다.
- 월 기본 목록 또는 비검색 필터 흐름은 기존 CSV 파일명과 동작을 유지했다.

### 스타일 정리

- `.csv-scope-hint` 스타일을 추가해 CSV 범위 안내를 필터 영역 안에 작게 표시하도록 했다.

## 4. 아직 부분 완료인 항목

### 검색 조건 전체 CSV

- 현재 CSV는 화면에 로드된 현재 페이지 거래만 내보낸다.
- 전체 검색 조건의 모든 페이지를 CSV로 내려받으려면 백엔드에 검색 조건 기반 export API를 별도로 추가하는 편이 안전하다.

### 영수증 파일 정리

- 거래 삭제 또는 영수증 삭제 이후 실제 첨부 파일 정리 정책 고도화가 아직 남아 있다.
- 저장소 파일과 DB 메타데이터 불일치 복구 정책도 추가 검토가 필요하다.

### mutation UX

- mutation 실행 중 중복 제출 방지와 pending 표시가 아직 화면별로 통일되어 있지 않다.
- API 재시도와 화면 상단 오류 배너도 아직 미완료다.
